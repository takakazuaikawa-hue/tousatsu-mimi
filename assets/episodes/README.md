# エピソードタイトル一枚絵

各エピソード開始時に **フルスクリーン背景画像** として表示されます。
画像を配置すると自動でカードの背景に反映され、暗いオーバーレイ越しに長文タイトルが読める形になります。

## ファイル名（完全一致）

| ファイル名 | エピソード | 背景シーン |
|---|---|---|
| `rico_tutorial.png` | 第1話 | バニー更衣室（鏡の前のミミ + リコ先輩） |
| `polka.png`         | 第2話 | 初心者向けポーカー卓（ポルカ vs ミミ） |
| `selina.png`        | 第3話 | 常連向け落ち着き卓（セリナ静かにベット） |
| `grano.png`         | 第4話 | 商人風豪華卓（グラーノが商人笑顔） |
| `velvet.png`        | 第5話 | VIPルーム（ヴェルベットがミミを見下ろす） |
| `ending.png`        | 第6話 | リゾートテラス夜景（ミミがチップを抱えて休憩） |

## 推奨サイズ・仕様

- **サイズ**：1920×1080（16:9）
- **形式**：PNG または JPG
- **解像度**：高解像度ほど豪華に見える
- **構図のポイント**：
  - 中央〜下部に余白を残す（タイトルカードが中央に重なるため）
  - 重要なキャラ顔は左右どちらか or 上部に
  - 暗めの画像でもOK（タイトル側で半透明黒のオーバーレイが乗る）

## 画像未配置時のフォールバック

画像が無い場合は、エピソード毎のCSS背景グラデーション（カジノラウンジ風など）が代用されます。テキストだけでも問題なく読めます。

## 表示の挙動

1. ゲーム側が `assets/episodes/{key}.png` を試行ロード
2. 成功 → 画像がフルスクリーン背景として表示、カードに `has-art` クラスが付き半透明黒背景に
3. 失敗 → CSS フォールバックグラデーションのみ表示（warning コンソール出力なし）
