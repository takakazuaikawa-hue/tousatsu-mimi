# 闘札圧倒伝ミミ

転生したらバニーガールだった私の外れスキル《ぱにゅぱにゅ》だけがレベルアップな件 — 心理ポーカーで成り上がる異世界カジノADV。

ブラウザだけで遊べる、心理バトル付き1人用ポーカーゲームです。

## 遊び方

GitHub Pages で公開されている場合：

👉 https://USERNAME.github.io/REPONAME/

ローカルで遊ぶ場合：

```bash
# このリポジトリをクローン
git clone https://github.com/USERNAME/REPONAME.git
cd REPONAME

# シンプルなHTTPサーバーで起動（Python 3 の場合）
python -m http.server 8000

# ブラウザで http://localhost:8000/ を開く
```

> ⚠️ `file://` で直接 `index.html` を開くと一部機能（テンプレート読み込み等）が動作しないことがあります。必ずHTTPサーバー経由で開いてください。

## ゲーム内容

- **Stage 1** リコ先輩のチュートリアル（操作・心理バトル基本）
- **Stage 2** ポルカ戦（ブラフ入門）
- **Stage 3** セリナ戦（ボード危険度・ベットサイズ）
- **Stage 4** グラーノ戦（ポットオッズ・割に合う判断）
- **Stage 5** ヴェルベット戦（VIPルーム・証拠突きつけ・ブラフブレイク）
- エンディング

各ステージ初回クリアでコイン報酬、戦術ノート自動解放。交換所でぱにゅぱにゅ強化・カード裏スキン購入。

### 特徴

- **心理バトル**：相手のセリフ・ベット・場札から違和感を読み取る
- **ぱにゅぱにゅ**：ミミの外れスキル。ヒントになる
- **ゾゾゾゲージ**：相手の動揺を可視化。MAXで「ブラフブレイク」
- **圧倒モード**：優勢時に自動勝利演出
- **闘札大逆転**：ボス戦の必殺演出
- **エピソード演出**：各話開始に長文ラノベ風タイトルカード

## 技術構成

- HTML / CSS / JavaScript（バニラ、フレームワーク不使用）
- ブラウザ単体で動作（サーバー不要）
- localStorage で進行保存
- 固定キャンバス 1280×800 を JS でウィンドウフィット拡縮
